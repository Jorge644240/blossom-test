# Blossom Test

A simple Node.js application that fetches data from public APIs (Pok√©mon and Digimon) and logs the requests to an in.memory mocked database. The application is built using TypeScript and follows a hexagonal architecture approach.

## Prerequisites

- Docker
- Docker Compose

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jorge644240/blossom-test
   cd blossom-test
   ```
2. Run the application using Docker Compose:
   ```bash
   docker-compose up --build
   ```

## Usage

The application will start a server on port 3000.
